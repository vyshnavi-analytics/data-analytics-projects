# Employee Analytics using SQL
 
## Project Overview
This project focuses on analyzing employee data using SQL to generate insights related to salary trends, performance, and department-wise analysis. The objective is to support HR and business decision-making through data-driven insights.

## Dataset
The dataset contains employee-level information such as 
 - Department
 - Role
 - Salary
 - Experience
 - Performance rating
 - Joining date

## Business Questions 
- Which department has the highest average salary?
- Who are the top-performing employees?
- How does experience impact employee salary?
- What is the salary distribution across departments?

## SQL Concepts Used
- SELECT, WHERE, GROUP BY
- Aggregate functions (AVG, COUNT)
- Window functions (RANK)
- CASE statements

## Key Insights
- The IT department has the highest average salary.
- Employees with higher experience tend to earn higher salaries.
- High-performing employees are concentrated in technical roles.
