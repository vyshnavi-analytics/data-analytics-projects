# Employee Analytics using SQL

## Project objective
Analyze employee data to extract insights related to salary trends,
department-wise performance, and employee demographics using SQL.

## Dataset
Employee table containing:
- Employee ID
- Name, Age, Gender
- Department, City
- Salary
- Joining Date
- Performance Rating

## SQL Concepts Used
- SELECT, WHERE
- CASE WHEN
- GROUP BY, HAVING
- Subqueries
- Aggregate Functions

## Key Insights
- Employees were classified into salary categories using CASE WHEN
- Department-wise average salaries were analyzed
- Departments paying above company average were identified
